# DevCamper API

> Backend API for DevCamper app, a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/setting on your own

## Install dependencies

```
npm install
```

## Run App
```
# run in dev mode
npm run dev

# Run in production mode
npm start
```
